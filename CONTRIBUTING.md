# Contributing to Spec-Driven AI Coding Harness

Thank you for your interest in contributing! This project helps teams turn vague requirements into structured specs for AI-assisted delivery.

## Development Setup

```bash
git clone https://github.com/yourusername/spec-driven-ai-coding-harness.git
cd spec-driven-ai-coding-harness
pip install -e ".[dev]"
```

## Code Style

- Follow PEP 8
- Use type hints
- Run `black` and `ruff` before committing
- All tests must pass

## Running Tests

```bash
pytest
```

## Submitting Changes

1. Fork the repository
2. Create a feature branch
3. Make your changes with tests
4. Ensure all tests pass
5. Submit a pull request
