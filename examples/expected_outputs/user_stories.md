# User Stories

## US-001: Upload attachments

As a business user,
I want to upload attachments to business documents,
so that supporting materials can be stored and reviewed later.

## US-002: Preview attachments

As a user,
I want to preview uploaded attachments,
so that I can verify file content without downloading it.

## US-003: Download attachments

As a user,
I want to download attachments,
so that I can access files offline or share them externally.

## US-004: Print attachments

As a business user,
I want to print attachments,
so that physical copies can be produced for signing, filing, or distribution.

## US-005: Delete attachments

As a business user,
I want to delete attachments,
so that outdated or incorrect files can be removed to keep the system clean.

## US-006: Record audit logs

As a system administrator,
I want the system to record audit logs for each attachment operation,
so that a complete audit trail is available for accountability and security investigation.
