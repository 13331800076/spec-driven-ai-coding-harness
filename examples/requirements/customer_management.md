# Requirement: Customer Management

The system should provide a comprehensive customer management feature for the sales and support teams.

Users should be able to create, update, view, and delete customer records.

The system should support customer classification (VIP, regular, prospect) and tagging.

The system should record the account manager assigned to each customer and the last contact date.

The system should support customer search by name, phone, email, or tag.

The system should enforce data validation: phone numbers must be valid, email must be unique.

The customer service should expose APIs for CRUD operations and search.
